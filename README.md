# Implementation of matrix algorithms and linear systems resolution using Python
# Computational Linear Algebra Projects

## Overview
This repository contains the implementation of fundamental linear algebra concepts applied to computational problems. Developed as part of the *Computational Linear Algebra* course at the Universidad de Buenos Aires (UBA).

## Technical Focus
* **Matrix Operations:** Efficient handling of vector spaces and matrix transformations.
* **Algorithm Implementation:** Translation of mathematical definitions (theorems/lemmas) into executable Python code.
* **Numerical Stability:** Analysis of computational errors and floating-point arithmetic.

## Key Projects
* **Project 1:** Algorithms for matrixes.
    * Applied rigorous mathematical logic to solve linear problems.

## Why It Matters
Deep understanding of linear algebra is essential for researching Neural Network architectures and optimization landscapes, aligning with the core technical requirements for AI research.
